In JetBrains Flex IDE, you can compare:

  • Directories
  • A class or a class member with another class or class member
  • A local file with its versions in the local history
  • A local file with its current revision in your version control repository
  • Any local file with the current file in the editor
  • Any two local files to one another (text or binary)
  • A text fragment selected in the editor with the clipboard contents

You can configure the differences tool usage in the Settings dialog (File | Settings | IDE: General: Diff options).

Use context menus to invoke comparison commands. Refer to the Comparison section of online help for details.
