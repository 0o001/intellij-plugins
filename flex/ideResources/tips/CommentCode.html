You may comment/uncomment lines and blocks of code using &shortcut:CommentByLineComment; and &shortcut:CommentByBlockComment;.

&shortcut:CommentByLineComment; comments/uncomments the current line or selected block with single line comments ("//...").
&shortcut:CommentByBlockComment; encloses the selected block with a block comment ("/*...*/").

To uncomment a block comment press &shortcut:CommentByBlockComment; anywhere inside it.
