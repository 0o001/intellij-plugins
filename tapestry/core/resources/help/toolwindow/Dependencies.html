Dependencies

This tab shows the Tapestry elements that the selected element uses (on tab left side).
This includes:
  • Embedded components
  • Injected pages
And the information of selected tree element (on tab right side).
This information includes:
  • Element description
  • Element property name
  • Binding parameters (only to embedded components)
  • Required and optional parameters (only to embedded components)
  • Method events declared in the class (only to embedded components)


The node Embedded components have all the Embedded components declared in the class and in the templates, the Embedded components in templates are in folders with the name of template were they are embedded.
In this tab you can use all the option declared in Tapestry MetaData, with right click in Tapestry component.



Navigate to Element Navigates to the selected element class.
Navigate to Usage Navigates to the place where the selected element is being used.
