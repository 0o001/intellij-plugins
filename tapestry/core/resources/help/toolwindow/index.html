Tapestry ToolWindow

This toolwindow shows contextualized information about some Tapestry element.

This section covers the following features:
  • Documentation
Documentation

This tab shows the documentation of a Tapestry component, page or mixin.
The documentation is composed of (where applicable):
  • General description
  • List of required parameters (with their name, prefix, default value and description)
  • List of optional parameters (with their name, prefix, default value and description)
  • List of artifacts
  • Examples
  • Notes



When the element selected it's not an Tapestry component, page or mixin the documentation is composed of:
  • List of Tapestry Applications
  • List of Core Library components, pages and mixins
