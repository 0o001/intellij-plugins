Tapestry Support for IDEA

This plugin adds Tapestry support in IDEA.

This plugin is designed to ease development of Tapestry projects in IntelliJ IDEA. It allows you to quickly setup a running project and provides several types of assistance during development that will help you work faster and better.

This documentation covers the following features:
  • Tapestry View
    • Drag and Drop
    • Safe Delete
  • Tapestry ToolWindow
  • Wizards
  • Template Editing
    • Syntax Highlighting
    • Error Checking
    • Auto-Completion
    • Navigation
