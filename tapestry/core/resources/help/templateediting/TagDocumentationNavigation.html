Quick consultation of component information

The plugin adds Tapestry specific features to the default template editor. This helps you understand your templates and also discover errors faster. One of these features is quick consultation of the information of a component.

The section covers the following issue:
  • Quick Documentation
Quick Documentation

To consult the documentation of a component without using this plugin it's necessary to open a browser and then open the page of component documentation. This process is too slow and there is a need to have multiple windows open, which makes organization more difficult.

With this feature, the consultation of component information it's easy you only have to use the shortcut key Ctrl+Shift+D on the tag of component that you want to see the documentation. When you do this the Tapestry Toolwindow will show that information.
